## Quick Reference Commands

### For Backend

```bash
# In: C:\Users\user\EKIMINA-SERVER

# Install dependencies
npm install

# Build TypeScript
npm run build

# Run migrations (create tables)
npm run migrate

# Start development server
npm run dev

# Start production server
npm start
```

### For Database

```bash
# Open PostgreSQL CLI
psql -U postgres

# List databases
\l

# Connect to database
\c ekimina_db

# List tables
\dt

# Execute SQL query
SELECT * FROM users;

# Exit
\q
```
